## 项目介绍

面向高校大学生的伙伴匹配系统，帮助大家找到兴趣相同的伙伴，该项目为移动端项目，基于Spring Boot  后端 + Vue3 前端的全栈项目 ，该项目核心功能包括：用户登录、更新个人信息、按标签搜索用户、建房组队、推荐相似用户等功能。

# 技术选型

### 前端

- Vue 3
- Vant UI 组件库
- Vite 脚手架
- Axios 请求库

## 后端

- Java SpringBoot 框架

- MySQL 数据库

- MyBatis-Plus

- MyBatis X 自动生成代码

- Redis 缓存

- Redis 分布式登录

- Redisson 分布式锁

- Easy Excel 数据导入

- Spring Scheduler 定时任务

- Swagger + Knife4j 接口文档

- Gson：JSON 序列化库

- 相似度匹配算法
